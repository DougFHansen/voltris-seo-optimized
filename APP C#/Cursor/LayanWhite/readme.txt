=== Layan White Cursor Set ===

By: nibbler (http://www.rw-designer.com/user/55973)

Download: http://www.rw-designer.com/cursor-set/layan-white

Author's description:

The original cursor theme, Layan White, converted into Windows format cursors using [https://pypi.org/project/win2xcur/ win2xcur]. Very important and great tool, go check it out! (Make sure you have Windows Subsystem for Linux installed and that you have the required dependencies.) :)

The progress.ani had one frame that was messed up, so I fixed that.
